## Installation Steps

### Option 1: Via Kiro IDE (Recommended)

1. **Open Kiro IDE**
2. **Open Command Palette** (Ctrl+Shift+P or Cmd+Shift+P)
3. **Search for:** "Open Kiro Hook UI" or "Powers"
4. **Click:** "Configure Powers" or "Manage Powers"
5. **Search for:** "Build a Power"
6. **Click:** Install
7. **Restart Kiro IDE** when prompted

### Option 2: Manual Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kiro-community/kiro-powers-power.git
   ```

2. **Navigate to the power:**
   ```bash
   cd kiro-powers-power/kiro-powers
   ```

3. **Follow the README.md** in that directory

## Troubleshooting

### Power Not Showing in Kiro IDE

1. **Restart Kiro IDE completely**
2. **Clear Kiro cache:**
   - Windows: `%APPDATA%\Kiro`
   - Mac: `~/Library/Application Support/Kiro`
   - Linux: `~/.config/Kiro`
3. **Reinstall the power**

### Installation Fails

1. **Check internet connection**
2. **Verify GitHub access**
3. **Try manual installation** (Option 2 above)
4. **Check Kiro IDE version** (should be latest)
