# Technology Stack

## Core Technologies
- **Frontend Framework**: React 18.2.0
- **Build Tool**: Vite 4.4.5
- **Styling**: Tailwind CSS 3.3.3
- **Routing**: React Router DOM 6.14.0
- **Icons**: Lucide React 0.263.1
- **Language**: JavaScript (JSX) with TypeScript types available

## Build System
- **Bundler**: Vite with React plugin
- **CSS Processing**: PostCSS with Autoprefixer
- **Package Manager**: npm (package-lock.json present)

## Development Setup
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Configuration
- **Vite Config**: Standard React setup with @vitejs/plugin-react
- **Tailwind Config**: Configured for all JS/JSX/TS/TSX files in src/
- **PostCSS**: Configured with Tailwind and Autoprefixer
- **TypeScript**: Types available for React components

## Key Dependencies
- React ecosystem (react, react-dom, react-router-dom)
- Tailwind CSS for utility-first styling
- Lucide React for consistent iconography
- Vite for fast development and optimized builds
