# Build a Power - Custom Power Creation

**Purpose:** Create custom Kiro Powers from APIs, services, or any external system to extend functionality.

**Source:** https://github.com/kiro-community/kiro-powers-power

**Status:** To be installed after Kiro IDE restart

---

## Overview

The "Build a Power" power allows you to create custom powers that integrate with:
- REST APIs
- GraphQL endpoints
- Database connections
- File systems
- Command line tools
- Any external service

This is particularly useful for your AI Receptionist SaaS to create custom integrations for:
- PayPal billing automation
- Twilio phone number management
- Custom client onboarding workflows
- Analytics and reporting
- Third-party CRM integrations

---

## Installation

1. Open Kiro Powers configuration panel
2. Search for "Build a Power" or use URL: https://github.com/kiro-community/kiro-powers-power
3. Install the power
4. Restart Kiro IDE if needed
