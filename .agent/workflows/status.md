# Project Status Workflow

Run this workflow to see current project status.

## Steps

1. Read the project context skill at `.agent/skills/project-context/SKILL.md`

2. Check the current spec status in `.kiro/specs/member-portal-billing/tasks.md`

3. Report:
   - Current phase
   - Completed tasks
   - Next tasks to work on
   - Any blockers

4. Ask user which task they want to work on next
